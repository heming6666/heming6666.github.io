      《The Clean Coder》 阅读记录 #

      13、团队与项目 #

      • 有凝聚力的团队：团队组建、克服差异（一起搞定一切）、多个项目中整体移动
      • 给足时间，成为引擎，军团

      14、辅导、学徒期与技艺 #

      • 导师很重要，可以更好、更加顺利地跨过一些坎。
      • 可以有多种形式的辅导。
      • 软件学徒期是必要的，类比医学生（人命关天）
      • 技艺：价值观、原则、技术、态度。非理性决策，非感情用事，只要能被人观察到，便具有传染性：成为表率，向人展示即可。
