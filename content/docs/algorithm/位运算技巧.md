# 位运算技巧

## 1、转换操作
1. 转小写
```C++
('a' | ' ') = 'a'
('A' | ' ') = 'a'
```

2. 转大写
```C++
('b' & '_') = 'B'
('B' & '_') = 'B'
```

3. 互换
```C++
('d' ^ ' ') = 'D'
('D' ^ ' ') = 'd'
```

## 2、正负号一样？
```C++
int x = -1, y = 2;
bool f = ((x ^ y) < 0); // true
int x = 3, y = 2;
bool f = ((x ^ y) < 0); // false
```

## 3、`n&(n-1)` 的应用
用于消除n的二进制表示中的最后一个1。
1. 位1的个数 
```C++
while(n!=0){ n=n&(n-1); res++;}
```
2. 判断一个数是不是 2 的指数
```C++
return(n&(n-1))==0;
```
